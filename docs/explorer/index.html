Residue view

Off by default — keeps the cancer/autophagy/mTOR protein graph legible. Turn on to see ~78 residue entities + 8 motif hyperedges across trypsin, TFIIIA, insulin, myoglobin, and GFP. Catalytic triads (arity-3), zinc fingers (arity-4), and helices (arity-16) are intrinsically N-ary — that's the case where a binary-edge database would need to reify every motif as a dummy node.

Fetch from AlphaFold-DB

Pulls globalMetricValue, gene, organism, and sequence length from alphafold.ebi.ac.uk, derives the pLDDT bucket using AF-DB thresholds, and commits a protein entity to this nDB. Skips if the accession is already loaded.
