Demos

Live, interactive demonstrations of nDB. Each demo gets its own URL path under this domain and its own narrative page explaining the data model it exercises.

Available now

langgraph_ndb — GraphRAG knowledge graph (the query-side demo)

A CC0 OpenAlex citation graph — 2,500 ML/transformer papers and 10,466 authors, ingested Rust-native straight into nDB. Where the other demos showcase how nDB stores N-ary facts, this one showcases how nDB answers: recursive CITES ancestry (the relation+ traversal that beats WITH RECURSIVE), vector kNN semantic search over abstract embeddings, and MVCC time-travel — scrub the year slider and watch the field grow from a handful in 2012 to 2,500 by 2026 — and semantic zoom that shows field "galaxies" zoomed out and dissolves them into papers as you zoom in. Plus N-ary AUTHORED hyperedges. An interactive 3D graph with a semantic-layout toggle (position = meaning).

alphafold_ndb — structural biology browser

20 cancer / autophagy / mTOR-signalling proteins seeded with real AlphaFold pLDDT values. 5 showcase proteins with per-residue + per-atom entities and structural-motif hyperedges (catalytic triads, zinc fingers, disulfide bonds, α-helices, β-sheet pairs). Bidirectional click sync between the 3D structure and the nDB hypergraph. Live AlphaFold-DB fetch — paste any UniProt accession.

exoplanet_ndb — N-ary discovery facts

91 confirmed exoplanets from NASA Exoplanet Archive across all 11 recognised detection methods (pulsar timing 1992 → Gaia astrometry 2025) and 40 stellar systems. Every discovery is one arity-4 hyperedge [star, planet, method, mission]. Every system is one arity-(N+1) hyperedge: TRAPPIST-1 = arity 8, Kepler-90 = arity 9. Live NASA TAP fetch — type a planet name to add it.

Read about it   Launch live

seismic_ndb — aftershock sequences

4,774 earthquakes from USGS: a 30-day live snapshot plus 13 curated historic mega-quake aftershock windows (1989 Loma Prieta through 2025 Myanmar). Each aftershock sequence is one arity-(N+1) hyperedge — the 2010 Maule (Chile) sequence holds 1,000 events in a single record. 14 major fault systems mapped, with binary ON_FAULT hyperedges linking events to their nearest fault. Live polling of the USGS past-hour feed.

biodiv_ndb — ecological interactions with temporal qualifiers

145 species with live Wikipedia photos, 22 regions, 57 binary/ternary interactions (pollination, mutualism, parasitism, predation), and 6 whole-ecosystem food-web hyperedges. The Yellowstone food web fits 17 species + the ecosystem into one arity-18 record. Each interaction carries a temporal qualifier — May–July for yucca/yucca-moth, year-round for hummingbird/heliconia — surfaced as a 12-month validity strip in the inspector. The angle prior demos hadn't shown: facts that are only true at certain times.

chemistry_ndb — reactions + living data

~43 curated reactions across organic synthesis, inorganic chemistry, and two full metabolic pathways (glycolysis 10-step linear, Krebs cycle 8-step). Each reaction is one arity-N hyperedge; stoichiometry is encoded as repeated role-fillers (six CO₂ molecules → six ROLE_REACTANT slots). Polyethylene polymerisation tops the arity charts at 1,003 (1,000 ethylene + 1 polymer + catalyst + solvent in one record).

Living data: middle pane splits into editor + live mirror. Edit a reaction's temperature on the LEFT → ΔG and K_eq recompute → commit to nDB → RIGHT pane sees the change arrive via the engine's /subscribe long-poll and pulses every related reaction.

Naming convention

Each demo lives at ndb.nextstar-erp.com/<demo_name>/. Its narrative page lives at ndb.nextstar-erp.com/demos-<demo_name>.html. Demos are independent — each can be running, paused, or replaced without affecting the knowledge site or other demos.
