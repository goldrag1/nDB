Feedback inbox

Every submission from the floating "Feedback" button on any page on this site is stored as one nDB entity (type_id = 200) in the dedicated feedback engine at 127.0.0.1:8744. Status changes append a new version — the engine is MVCC, so prior states stay in the audit trail.
