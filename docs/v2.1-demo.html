nDB v2.1 demo — protein complexes

A small biology dataset (15 proteins, 6 protein complexes) seeded into a real nDB database, then visualised through the three v2.1 N-dim renderers. Hover any node or polygon for metadata.

Source database: /tmp/v21-demo-ndb · rendered by cargo run -p ndb-renderer --example v21_demo

1 · Hypergraph diagram (§2.12) — the showcase

Entities are labelled nodes; each protein complex is a polygon connecting its members. This is what a hyperedge looks like — a single fact connecting N entities, not a binary edge with reified properties. Hover any node or polygon for properties.

Node colour = entity type · polygon colour = hyperedge identity · layout = deterministic Fruchterman-Reingold

2 · Parallel coordinates (§2.11) — proteins across discovery year × function

One axis per dimension; each protein becomes a polyline crossing every axis. Useful for spotting clusters + outliers in N-dim numeric/categorical data. Colour = function class.

3 · Pivot table (§2.10) — protein × (pathway / complex) membership

Compound column labels (pathway / complex) expand 4-5 dimensional data into a single tabular view. Empty cells = no membership.
