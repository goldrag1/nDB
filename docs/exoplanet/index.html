Hits the NASA Exoplanet Archive TAP service for one row, then commits one PLANET entity + one arity-4 DISCOVERY hyperedge. Star, method, mission entities are reused if already present.

What the demo proves

Every dot in the scatter is one PLANET entity. Every discovery is one arity-4 hyperedge: [star, planet, method, mission]. Every multi-planet system is one arity-(N+1) hyperedge. Kepler-90 = arity 9. TRAPPIST-1 = arity 8. Inspect the right pane to see them as real nDB records.
